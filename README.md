Big Chefs AI Waste Management Simulation

Bu proje, restoranlar için gıda israfını optimize eden bir simülasyon motorudur. 
Özellikle **90 günlük veri üretimi** ve **%90 tahmin başarısı** hedeflenmiştir.

## Nasıl Çalıştırılır?
1. Önce gerekli kütüphaneleri kurun: `pip install -r requirements.txt`
2. Sonra kodu çalıştırın: `python main.py`
3. Analiz için `BigChefs_90Days_Final_Data.xlsx` dosyasına bakın.

## Teknik Detaylar
- **Model:** Cold-start fazı simülasyonlu öngörücü analitik.
- **Hedef:** Yıllık ~$240k tasarruf.
